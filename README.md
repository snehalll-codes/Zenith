#Zenith — Student Dashboard
Zenith is a student productivity dashboard built for focused, organized, and intentional learning. Designed with simplicity and purpose, it brings everything a student needs into one clean interface.
Features

#Login & Signup — Personalized experience for every user
Task Manager — Track tasks across Pending, In Progress, and Done
Study Timer — Focus sessions with a live progress ring and study log
Breathing Exercise — Guided 4-step breathing cycle for stress relief
Mini Games — Memory match and word scramble to relax your mind
Riddles — Quick brain exercises to stay sharp
Journal — Daily entries with mood tracking and delete support
Daily Quote — A fresh motivational quote every day

#Tech Stack
Built using HTML, CSS, and JavaScript — no frameworks, just fundamentals.
About
Zenith was built from scratch by a student, for students. Every feature exists because it was actually needed — not just added for the sake of it.
